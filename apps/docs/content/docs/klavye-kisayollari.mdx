Wingman, hızlı navigasyon ve verimli kullanım için kapsamlı klavye kısayolları sunar.

## Genel Kısayollar

| Kısayol | İşlev |
|---------|-------|
| <Kbd keys="Ctrl+K" /> | Komut Paleti (Command Palette) |
| <Kbd keys="Escape" /> | Açık paneli veya dialog'u kapat |

---

## Komut Paleti

<Callout type="info" title="En Güçlü Aracın">
  Komut Paleti, Wingman'in en güçlü navigasyon aracıdır. <Kbd keys="Ctrl+K" /> ile aç ve fare kullanmadan her yere ulaş.
</Callout>

Komut paleti ile yapabileceklerin:

<div className="grid grid-cols-1 md:grid-cols-3 gap-3 my-6">
  <FeatureCard icon="search" title="Sayfa Arama">
    Herhangi bir sayfaya hızlıca git — Dashboard, Geçmiş, Ayarlar...
  </FeatureCard>
  <FeatureCard icon="zap" title="Hızlı Eylemler">
    Yeni mülakat başlat, ayarlara git, tema değiştir.
  </FeatureCard>
  <FeatureCard icon="history" title="Son Mülakatlar">
    Son mülakatlarına hızlı erişim ve devam etme.
  </FeatureCard>
</div>

## Mülakat Odası Kısayolları

### Kod Editörü

| Kısayol | İşlev |
|---------|-------|
| <Kbd keys="Ctrl+Enter" /> | Kodu çalıştır |
| <Kbd keys="Ctrl+S" /> | Kodu kaydet |
| <Kbd keys="Ctrl+Space" /> | Otomatik tamamlama |
| <Kbd keys="Ctrl+F" /> | Kod içinde bul |
| <Kbd keys="Ctrl+H" /> | Bul ve Değiştir |
| <Kbd keys="Ctrl+/" /> | Satırı yorumla / yorumu kaldır |
| <Kbd keys="Alt+↑" /> | Satırı yukarı taşı |
| <Kbd keys="Alt+↓" /> | Satırı aşağı taşı |
| <Kbd keys="Ctrl+D" /> | Kelimeyi seç (çoklu seçim) |
| <Kbd keys="Ctrl+Shift+K" /> | Satırı sil |

### Genel

| Kısayol | İşlev |
|---------|-------|
| <Kbd keys="Ctrl+Z" /> | Geri Al |
| <Kbd keys="Ctrl+Shift+Z" /> | İleri Al |
| <Kbd keys="Ctrl+A" /> | Tümünü Seç |

---

## Navigasyon Kısayolları

Komut paletini açtıktan sonra (<Kbd keys="Ctrl+K" />) şu kısayolları kullanabilirsin:

| Tuş | Hedef |
|-----|-------|
| `d` | Dashboard'a git |
| `n` | Yeni mülakat başlat |
| `q` | Soru bankasına git |
| `h` | Geçmişe git |
| `p` | İlerleme sayfasına git |
| `s` | Ayarlara git |

<Callout type="tip" title="Verimlilik">
  Komut paletini düzenli kullan — fare kullanmadan her yere ulaşabilirsin. Bu alışkanlık gerçek iş ortamında da işine yarar!
</Callout>
