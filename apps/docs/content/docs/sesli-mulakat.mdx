Wingman'in en güçlü özelliği **gerçek zamanlı sesli mülakat deneyimidir**. Gerçek bir mülakatçıyla konuşur gibi, Türkçe olarak sesli iletişim kurarsın.

## Ses Pipeline'ı

<VoicePipelineDiagram />

<Steps>
  <Step number={1} title="Dinleme (STT)">
    Mikrofonundan gelen ses **Freya STT** modeli ile metne çevrilir. fal.ai altyapısında çalışan bu model, Türkçe konuşmayı yüksek doğrulukla tanır.
  </Step>
  <Step number={2} title="Düşünme (LLM)">
    Metin AI mülakatçıya gönderilir. OpenRouter üzerinden güçlü bir LLM, mülakat bağlamına uygun yanıt üretir.
  </Step>
  <Step number={3} title="Konuşma (TTS)">
    AI'ın yanıtı **Freya TTS** modeli ile doğal Türkçe sese çevrilir ve sana oynatılır.
  </Step>
</Steps>

---

## Freya — Türkçe Ses Modeli

<div className="grid grid-cols-1 md:grid-cols-2 gap-3 my-6">
  <FeatureCard icon="mic" title="Freya STT">
    Türkçe konuşma tanıma (Speech-to-Text). Yüksek doğruluk, düşük gecikme.
  </FeatureCard>
  <FeatureCard icon="volume" title="Freya TTS">
    Doğal Türkçe ses sentezi (Text-to-Speech). İnsan benzeri telaffuz ve tonlama.
  </FeatureCard>
</div>

```ts title="Freya fal.ai entegrasyonu"
// STT — Ses → Metin
const transcript = await fal.subscribe("freya-mypsdi253hbk/freya-stt", {
  input: { audio_url: audioUrl, language: "tr" },
  path: "/generate",
});

// TTS — Metin → Ses
const audio = await fal.subscribe("freya-mypsdi253hbk/freya-tts", {
  input: { input: "Merhaba, mülakata başlayalım!", response_format: "wav" },
  path: "/generate",
});
```

## Sesli İletişim İpuçları

<Callout type="tip" title="Net Konuş">
  AI mülakatçı net ve anlaşılır konuşmayı daha iyi anlar. Yavaş ve düzgün konuş.
</Callout>

<Callout type="tip" title="Düşünme Süresi Al">
  "Bir saniye düşüneyim" diyerek düşünme süresi alabilirsin — gerçek mülakatlarda da böyle yapılır!
</Callout>

<Callout type="tip" title="Tekrar İste">
  Soruyu anlamadıysan "Tekrar edebilir misiniz?" diyebilirsin. Mülakatçı soruyu yeniden açıklar.
</Callout>

## Metin Alternatifi

Sesli iletişim kuramadığın durumlarda mülakat panelindeki **mesaj kutusunu** kullanarak yazılı olarak da iletişim kurabilirsin.

```
Gürültülü ortam? → Metin moduna geç
Mikrofon sorunu? → Chat panelini kullan
```
