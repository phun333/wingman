Dashboard, Wingman'e giriş yaptığında karşına çıkan ana sayfadır. Tüm önemli bilgiler ve hızlı erişim butonları burada bulunur.

## Genel Bakış Kartları

Dashboard'un üst kısmında üç temel metriği görebilirsin:

<div className="grid grid-cols-1 md:grid-cols-3 gap-3 my-6">
  <FeatureCard icon="target" title="Toplam Mülakat">
    Bugüne kadar yaptığın toplam mülakat sayısı.
  </FeatureCard>
  <FeatureCard icon="check" title="Tamamlanan">
    Başarıyla tamamladığın mülakatlar.
  </FeatureCard>
  <FeatureCard icon="calendar" title="Bu Hafta">
    Bu hafta yaptığın mülakat sayısı ve haftalık hedefin.
  </FeatureCard>
</div>

## Streak Takibi

<Callout type="fire" title="Günlük Streak">
  Her gün en az bir mülakat yaparak streak'ini devam ettir! Streak'in motivasyonunu yüksek tutar ve düzenli pratik alışkanlığı kazandırır.
</Callout>

Dashboard'da günlük streak sayacı gösterilir:

```ts title="Streak hesaplama mantığı"
// Streak = ardışık gün sayısı
// Her gün en az 1 mülakat yap
const streak = {
  current: 7,       // Mevcut ardışık gün
  longest: 23,      // En uzun seri
  todayDone: true,  // Bugün mülakat yapıldı mı?
};
```
