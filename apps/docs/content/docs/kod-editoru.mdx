<Callout type="tip" title="Önce Planla, Sonra Kodla">
  AI mülakatçıya yaklaşımını sesli olarak açıkla. Brute-force çözümden bahset, sonra optimize et.

<Callout type="tip" title="Düşünceni Seslendir">

</Callout>

## Örnek Kod

```ts title="Binary Search implementasyonu"
function binarySearch(nums: number[], target: number): number {
  let left = 0;
  let right = nums.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);

    if (nums[mid] === target) return mid;
    if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
  }

  return -1; // target bulunamadı
}

// Zaman karmaşıklığı: O(log n)
// Alan karmaşıklığı: O(1)
```
